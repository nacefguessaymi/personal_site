## About
Built with Hugo using the Blowfish theme.
Includes a custom animated hero background (Vanta.js)
that syncs with the site's light/dark color scheme,
and a citation-style publications section for future research write-ups.

## Tech stack
- Hugo — static site generator
- Blowfish — Hugo theme
- Vanta.js + Three.js — animated hero background
- Netlify — hosting & deployment
- Namecheap — domain registration


## Running locally

```bash
git clone --recurse-submodules https://github.com/nacefguessaymi/personal_site.git
cd personal_site
hugo mod tidy
hugo server
```
Site will be available at http://localhost:1313.
